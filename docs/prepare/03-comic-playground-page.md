# Poprako W 漫画页手册

## 页面定位

`/comic-playground` 是 Poprako W 最核心的业务页面。

如果说工作区偏“我的任务”，那么漫画页偏“团队内容管理中心”。它围绕作品集、漫画、章节、页面、分工、流程推进来组织。

适合把它理解成：

1. 团队漫画总入口。
2. 作品集管理页。
3. 漫画与章节的主要运营后台。

## 页面结构

本页主要分为三层：

1. 顶部筛选与工具区。
2. 左侧作品集侧边栏。
3. 主内容区。

主内容区又有两种视图：

1. 翻译视图。
2. 监修视图。

## 功能详解

### 1. 顶部筛选与工具区

顶部提供漫画检索和创建入口。

用户可以：

1. 输入标题、作者、序号关键词。
2. 按 `Enter` 触发搜索。
3. 按上传、翻译、校对、嵌字、监修、发布六个阶段筛选流程状态。
4. 点击创建漫画按钮，打开创建弹窗。
5. 在较小屏幕中点击侧边栏按钮，展开或收起作品集栏。

其中：

1. 上传、监修、发布是二态筛选：未完成、已完成。
2. 翻译、校对、嵌字是三态筛选：未完成、进行中、已完成。

### 2. 作品集侧边栏

作品集是漫画的上层分组容器。

侧边栏负责：

1. 展示当前团队下的作品集列表。
2. 切换当前激活作品集。
3. 展示每个作品集下漫画数量。
4. 打开新建作品集弹窗。

注意：虽然组件保留了删除作品集能力的回调，但当前界面里删除按钮还没有放出来，因此用户实际只能切换和新建，不能直接在 UI 中删除。

### 3. 翻译视图

翻译视图以卡片网格方式展示漫画，强调快速进入编辑。

特征包括：

1. 无限滚动加载。
2. 卡片方式显示漫画。
3. 点击卡片直接打开漫画详情弹窗。

### 4. 监修视图

监修视图会展示更偏进度检查的信息，适合监修/管理角色总览项目推进情况。

它同样支持：

1. 无限滚动。
2. 点击条目打开漫画详情。

## 漫画详情弹窗

本页中的漫画详情弹窗，是全站最复杂的业务操作面板之一。

### 可查看的信息

1. 漫画标题、作者、序号、封面。
2. 漫画下的章节列表。
3. 当前章节的页面列表。
4. 当前章节成员分工。
5. 当前章节所处流程状态。

### 可执行的章节操作

1. 新建章节。
2. 删除章节。
3. 选择章节。
4. 加载更多章节。

### 可执行的页面操作

1. 当章节还没有页面时，批量上传原图。
2. 支持拖拽上传与点击上传。
3. 上传时会按文件名自然顺序排序，尽量符合漫画页顺序。
4. 当章节已有页面时，对单页重传。
5. 在满足权限时清空整章页面后重传。
6. 点击页面缩略图进入翻译器。

### 可执行的数据操作

1. 导入章节数据。
2. 导出章节数据。
3. 导出章节 LP 数据。
4. 上传漫画封面。

### 可执行的分工操作

1. 查看当前章节各角色已分配成员。
2. 为角色添加成员。
3. 从角色移除成员。
4. 当前用户自己加入某个角色。
5. 当前用户自己退出某个角色。

### 可执行的流程操作

1. 根据当前章节状态推进工作流。
2. 不允许对已经推进过的状态重复推进。

### 可执行的导航操作

1. 正常进入翻译器。
2. 只读进入翻译器。

### 管理员专属高风险操作

1. 删除整部漫画。
2. 删除时章节和页面数据也会一并删除。

## 典型使用流程

### 场景一：创建一部新漫画

1. 进入 `/comic-playground`。
2. 在左侧切换到目标作品集。
3. 点击顶部加号创建漫画。
4. 在创建弹窗中填写漫画信息。
5. 创建成功后列表刷新。

### 场景二：为漫画新建章节并上传页面

1. 打开目标漫画详情。
2. 新建章节。
3. 切换到新章节。
4. 在空页面区域点击上传，或把图片拖入上传区。
5. 上传完成后生成页面缩略图。

### 场景三：把成员分配到章节角色

1. 打开漫画详情。
2. 选择章节。
3. 在分工区域点击某个角色的添加入口。
4. 在成员选择弹窗中检索并选定成员。
5. 分配成功后角色区刷新。

### 场景四：从漫画页进入翻译器

1. 打开目标漫画详情。
2. 选择章节。
3. 点击某一页缩略图。
4. 系统跳到 `/translator/:chapterId/:pageId`。
5. 若当前用户只有只读权限，会以只读模式进入。

## 页面限制与规则

1. 本页强依赖当前选中团队和当前激活作品集。
2. 没有激活作品集时，漫画列表会是空的。
3. 搜索输入不是实时生效，必须按 `Enter`。
4. 作品集删除能力暂未开放到界面。
5. 导入导出、上传原图、分配成员、删除漫画等均有权限门槛。
6. 章节页面为空时才能显示整章上传入口；有页面后以单页重传为主。

## 与其他页面的关系

1. 登录成功默认首先进入本页。
2. 本页和 `/workspace` 都能打开同一套漫画详情弹窗。
3. 进入翻译器时会带上 `returnTo=/comic-playground` 以及漫画、章节上下文。
4. 翻译器退出后可返回本页，并恢复原来打开的漫画上下文。
