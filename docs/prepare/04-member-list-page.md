# Poprako W 成员页手册

## 页面定位

`/member-list` 是汉化组成员总览与管理页。

它围绕“成员”而不是“漫画”组织内容，主要处理：

1. 成员查询。
2. 成员按职位筛选。
3. 邀请新成员。
4. 查看待处理邀请。
5. 管理员调整成员角色。

## 页面结构

页面结构比较直接：

1. 顶部筛选头部。
2. 成员列表区。
3. 邀请成员弹窗。
4. 成员详情弹窗。

## 功能详解

### 1. 成员搜索

用户可输入昵称关键词进行模糊搜索。

注意：

1. 搜索不是实时触发。
2. 需要按 `Enter` 才会提交搜索条件。

### 2. 按角色筛选

顶部提供一排职位快捷按钮：

1. 图。
2. 翻。
3. 校。
4. 嵌。
5. 美。
6. 监。
7. 传。

筛选规则：

1. 一次只能选一个职位。
2. 再点一次当前职位即可取消筛选。

### 3. 成员列表

成员列表使用滚动加载方式呈现。

用户可以：

1. 浏览成员卡片。
2. 查看昵称、头像等成员基础信息。

如果当前操作者是管理员：

1. 点击成员卡片会打开成员详情弹窗。

如果不是管理员：

1. 成员通常只能浏览，不能进入角色编辑。

### 4. 邀请新成员

点击右上角加号，可打开邀请弹窗。

邀请弹窗可执行的操作包括：

1. 输入被邀请人的 QQ 号。
2. 勾选要授予的角色。
3. 生成邀请码。
4. 复制邀请码给目标成员。
5. 查看当前所有待处理邀请。
6. 复制已有邀请码。
7. 撤销尚未使用的邀请码。

页面右侧的待处理邀请列表只显示 `pending` 状态的邀请码。

### 5. 成员详情与角色修改

管理员点击某个成员后，可以打开成员详情弹窗。

弹窗中可以：

1. 查看成员头像、昵称、QQ、最近活跃时间。
2. 查看是否具备管理员标识。
3. 勾选或取消各类角色位。
4. 提交保存角色变更。

## 典型使用流程

### 场景一：邀请新翻译入组

1. 进入 `/member-list`。
2. 点击右上角加号。
3. 填入对方 QQ。
4. 勾选“翻译”角色，必要时追加其他角色。
5. 点击“邀请”。
6. 复制生成的邀请码发送给对方。

### 场景二：筛选出所有校对成员

1. 进入 `/member-list`。
2. 点击“校”按钮。
3. 成员列表仅显示带校对身份的成员。

### 场景三：管理员调整某位成员权限

1. 进入 `/member-list`。
2. 点击目标成员卡片。
3. 在详情弹窗中切换角色位。
4. 点击确认提交。

## 页面限制与规则

1. 本页依赖当前选中的团队，没有团队时成员列表会为空。
2. 搜索要按 `Enter` 才触发。
3. 角色筛选是单选，不支持同时筛多个岗位。
4. 成员详情的角色编辑只对管理员开放。
5. 邀请界面前端未显式做管理员隐藏，但后端仍可能进行权限校验。

## 与其他页面的关系

1. 在设置页切换团队后，本页展示的成员也会切换到新团队。
2. 新成员拿到邀请码后，会在登录页完成注册。
3. 注册成功后，该成员就会成为当前团队可见成员。
