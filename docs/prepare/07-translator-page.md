# Poprako W 翻译器页面手册

## 页面定位

`/translator/:chapterId/:pageId` 是 Poprako W 的全屏编辑器。

这是整个系统里最强操作性的页面，承担章节页面级别的文本处理工作。

它支持三种工作状态：

1. 翻译模式。
2. 校对模式。
3. 只读模式。

## 进入条件

访问本页至少需要：

1. `chapterId`。
2. `pageId`。

如果缺少参数，页面会直接提示：

1. 缺少章节 ID。
2. 缺少页面 ID。

页面在真正渲染编辑器前还会检查登录态。如果用户尚未完成登录态恢复，会先尝试拉取用户信息；失败则跳转到 `/login`。

## 页面结构

进入成功后，页面由两大区域组成：

1. 画布区。
2. 侧边编辑区。

此外还有若干浮层和辅助面板：

1. 工具下拉菜单。
2. 页码翻页器。
3. 快捷键面板。
4. 特殊符号面板。
5. 保存失败确认框。
6. 删除文本块确认框。

## 核心功能

### 1. 加载章节页面

编辑器会先加载当前章节的全部页面，再根据 `pageId` 定位到起始页。

如果当前章节没有任何页面，会提示“当前章节暂无页面”。

### 2. 加载并编辑文本块

每一页会加载对应的文本块列表，也就是翻译单元。

可执行的编辑动作包括：

1. 新建文本块。
2. 选中文本块。
3. 调整文本块位置。
4. 修改文本内容。
5. 删除文本块。
6. 在列表中聚焦具体文本块。

如果某文本块已经有翻译内容或校对内容，删除前会要求二次确认。

### 3. 模式切换

模式能力与用户身份有关。

如果当前用户是本章节校对：

1. 可在翻译模式、校对模式、只读模式之间循环切换。

如果不是校对：

1. 通常不会拥有完整的多模式切换能力。
2. 某些入口会直接以只读模式进入。

### 4. 页面翻页

画布右上角的分页器支持：

1. 切换上一页。
2. 切换下一页。
3. 查看整章页数。
4. 看到每页的单元完成统计。

翻页前如果当前页有未保存修改，编辑器会尝试先保存。

### 5. 手动保存与自动保存时机

编辑器支持保存按钮，也会在一些动作前尝试冲刷未保存内容，例如：

1. 翻页时。
2. 退出时。
3. 从校对模式切换到只读模式时。

如果保存失败，会弹出确认框，允许：

1. 再次重试。
2. 放弃当前页未保存改动并继续。

### 6. 重定位与标记创建控制

状态栏中还提供两个细粒度开关：

1. 重定位开关。
2. 标记创建开关。

重定位开启后，选中文本块时画布会尽量居中到该位置。

标记创建关闭后，可阻止继续在画布中添加新标记。

### 7. 校对预览透明度

用户可以切换校对预览显示强度，用于在视觉上降低或恢复标记覆盖感。

### 8. 快捷键与特殊符号面板

在非只读状态下，工具菜单可打开：

1. 快捷键说明面板。
2. 特殊符号面板。

适合高频翻译时提高输入效率。

## 典型使用流程

### 场景一：从漫画详情进入翻译

1. 在工作区或漫画页打开漫画详情。
2. 选择章节。
3. 点击页面缩略图。
4. 进入 `/translator/:chapterId/:pageId`。

### 场景二：完成一页翻译并切到下一页

1. 在画布中调整和新增文本块。
2. 在右侧填写翻译内容。
3. 点击保存，或直接点击下一页。
4. 编辑器保存成功后进入下一页。

### 场景三：以只读方式检查内容

1. 从漫画详情中点击只读入口，或因权限不足以只读进入。
2. 浏览页面与文本块。
3. 不可执行编辑和新增操作。
4. 检查完成后点击退出。

## 返回逻辑

翻译器退出时会优先尝试返回来源页。

如果 URL 带有以下上下文：

1. `returnTo=/workspace` 或 `returnTo=/comic-playground`
2. `comicId`
3. `chapterId`

那么退出后会回到来源页，并尽量恢复原来打开的漫画与章节。

如果没有这些参数，就退回浏览器历史上一页。

## 页面限制与规则

1. 缺少 `chapterId` 或 `pageId` 时无法正常进入。
2. 当前章节没有页面时无法开始编辑。
3. 只读模式下不可编辑、不可新增标记、不可删除标记。
4. 是否允许切换到校对模式，取决于当前用户是否是章节校对。
5. 离开页面前若有脏数据，会优先尝试保存。

## 与其他页面的关系

1. 主要从 `/workspace` 和 `/comic-playground` 的漫画详情弹窗进入。
2. 退出后会尽量返回进入前的业务页面。
3. 本页本身不显示应用壳，不展示侧边栏与底部导航，是独立全屏工作环境。

## 白杨子可如何描述本页

“翻译器页是章节页面级别的全屏工作台，支持文本块标记、翻译、校对、只读审阅、翻页与保存，是 Poprako W 的核心生产页面。”
